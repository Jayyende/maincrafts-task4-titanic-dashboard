# 🚢 Titanic Data Visualization Dashboard

> **Maincrafts Technology – Data Science & Data Analysis with Python Internship (Task 4)**

## 📌 Project Overview

This project presents a comprehensive **Data Visualization Dashboard** built using the Titanic dataset. The primary objective is to transform raw passenger data into meaningful visual insights through effective data cleaning, feature engineering, and graphical analysis.

The dashboard demonstrates how various visualization techniques can be used to explore relationships between passenger characteristics and survival outcomes.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Handle missing values using appropriate techniques
* Create meaningful features for analysis
* Develop multiple visualizations to explore the dataset
* Interpret findings through data storytelling

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

---

## 📂 Dataset

**Dataset:** Titanic Passenger Dataset

The dataset contains passenger information such as:

* Passenger Class
* Age
* Gender
* Fare
* Family Information
* Embarkation Port
* Survival Status

---

## 📊 Dashboard Visualizations

This dashboard includes the following visualizations:

* 📈 Age Distribution Histogram
* 📊 Survival Rate by Passenger Class
* 📦 Fare Distribution using Box Plot
* 🔵 Age vs Fare Scatter Plot
* 🔥 Correlation Heatmap

---

## 📈 Key Insights

* Most passengers belonged to the young adult age group.
* First-class passengers showed significantly higher survival rates.
* Ticket fares varied greatly across passenger classes.
* The scatter plot highlighted relationships between passenger age, fare, and survival.
* The correlation heatmap revealed important numerical relationships within the dataset.
